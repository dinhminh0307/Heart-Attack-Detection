1. Install Jupyter Notebook.
2. Make sure 'Heart_Attack_Cleaned.csv' and all .ipynb files are in the same folder.
3. Be sure to be inside the directory of the files.
4. Run the command in terminal: jupyter notebook
5. Select a .ipynb file to be run.
6. Select 'Kernel' and afterwards 'Restart & Run All' to run all the code cells in the file.
